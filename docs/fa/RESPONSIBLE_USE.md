# استفاده مسئولانه

از این پروژه برای research، simulation، education و تحلیل آفلاین reusable استفاده کنید.

از آن برای ساخت automation مخرب، دست‌کاری بازار، نقض قوانین شبکه یا صرافی، افشای داده خصوصی یا تشویق رفتار مالی پرریسک استفاده نکنید. هر سیستم زنده‌ای که بر پایه این هسته ساخته شود به بازبینی مستقل حقوقی، امنیتی، مالی و عملیاتی نیاز دارد.

نتیجه‌ها فقط candidate تحلیلی هستند؛ نه trade instruction، نه advice مالی و نه guarantee سود.
