# مستندات فارسی

- `README_DETAILS.md` — توضیح تفصیلی درباره کاربران هدف، مدل استفاده و تناسب عملی
- `ADVANCED_README.md` — توضیح پیشرفته درباره زیرساخت، مدل اعتماد و توسعه‌پذیری
- `PUBLIC_RELEASE_GUIDE.md` — راهنمای metadata گیت‌هاب، جایگاه release و متن انتشار اول
- `README.md` — نمای کلی پروژه، هدف‌ها، غیرهدف‌ها و وضعیت بلوغ
- `ARCHITECTURE.md` — ماژول‌های اصلی، جریان داده، schemaها، مرز Rust و subsystem قراردادها
- `INSTALLATION.md` — نصب، راه‌اندازی و اعتبارسنجی اولیه
- `USAGE.md` — CLI، API پایتون، حالت validate-only و قالب snapshot
- `CONFIGURATION.md` — تنظیمات، پیش‌فرض‌های امن و overrideهای CLI
- `EXTENSION.md` — نقطه‌های توسعه برای provider، reporter، policy، زبان و قرارداد
- `SECURITY.md` — hygiene امنیتی، مدل امنیت قراردادها و مانع تاریخچه Git
- `RESPONSIBLE_USE.md` — کاربردهای مجاز و هشدارهای سوءاستفاده
- `DEVELOPMENT.md` — مسیر توسعه محلی و انتظارهای مشارکت
- `MIGRATION.md` — تغییرات نسبت به سیستم قبلی
- `HISTORY_CLEANUP.md` — سابقه پاک‌سازی Git history
- `rust/arbcore-rs/README.md` — analyzer اختیاری Rust

## Notice

نسخه انگلیسی در `docs/en/` منبع canonical برای تصمیم‌های release، runbook و جزئیات قراردادهای هوشمند است.
